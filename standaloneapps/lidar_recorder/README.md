# Lidar Recorder Tool

application to record icp corrected stitched point clouds. this generates combined lidar point cloud frame by frame

## Purpose

This tool:
1. Loads sensors from rig file
2. Runs ICP alignment between two lidars
3. Captures stitched points frame by frame.


## Output

The tool outputs .bin files for the captured frames
