# LiDAR Object Detection with ICP Free Space

This is a standalone server application for object detection and free space detection. see command.txt to see command to run applicaiton.

This application requires Eigen3 to be installed before building.

Install Eigen3 using:
```bash
sudo apt-get install libeigen3-dev
```

Download model.engine from here:
https://drive.google.com/file/d/1M3tB6sROT0V167R9OoYilq5f-HQtyO_L/view?usp=drive_link
